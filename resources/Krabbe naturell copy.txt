Tittel: Krabbe naturell
Undertittel: Kokt krabbekjøtt som smaker friskt av hav er en av de mest fantastiske sjømatrettene man kan servere. Server krabben med loff, smør, majones og sitron.

Ingredienser:
4 stk krabbe
0 loff
0 smør
0 majones
2 stk sitroner sitron
0 eddik
0 pepper


Fremgangsmåte:
Dersom du kjøper levende krabbe må den kokes først
Du kan lese hvordan du koker, renser og deler krabbe her
Krabben kan serveres som den er, men rens gjerne før servering ved å fjerne paven og gjellene, og knekke klørne med en nøtteknekker
Legg krabbekjøttet tilbake i skallet så krabben ser hel ut før du serverer den
Server krabbe naturell med loff, smør, majones og sitronbåter, med eddik og pepper på bordet


Tips: Som variasjon i tilbehøret kan du servere chilisaus til
