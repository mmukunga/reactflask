Tittel: Skalldyrfat
Undertittel: Skalldyrfatet er sommerens høydepunkt. Ferske skalldyr og godt tilbehør er enkelt og får opp stemningen. Beregn 500 gram skalldyr med skall per person.

Ingredienser:
0 reker, med skall
0 hummer, kokt
0 sjøkreps
0 krabbe


Fremgangsmåte:
Beregn ca
500g skalldyr per person
Server med tilbehør som loff, smør, sitron, lime, majones, aioli og dill
